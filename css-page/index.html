CSS

Cascading Style Sheets (CSS)

A language which dictates how HTML elements are to be displayed or styled.

Best practices

  • Comments: provide a brief explanation for label to the code
  • Accessibility:
    select fonts which are appropriate for low vision
    ensure sufficient contrast
    ensure spacing is appropriate for magnification
  • Formatting: order the code in a logical order and with flagging comments for each section
  • Validate the code: check the code for errors using a CSS validator
  • Make styling responsive to screen size

Hints

  • Check the spelling of each class
  • Check container levels
  • Check folder pathway
  • Check color and contrast
  • Check mobile site portrait and landscape
  • Google Fonts provides a range of free font options
  • Use media queries to make the webpage responsive to different screen sizes

Flexbox

Flexbox is a style of layout that allows containers to flexibly adjust to different screen sizes.

Parent properies

  • justify-content
  • align-items
  • flex-wrap
  • align-content
  • gap
  • flex-direction
  • flex-flow

Child properites

  • flex
  • flex-basis
  • flex-shrink
  • flex-grow
  • order
  • align-self

One & Done

  • Border: width color type;
  • Width and height can be measured in pixels or percentages or can be relative using flexbox
  • You can use calculations to set widths that adapt to screen size
  • Break-inside dictates how a column behaves inside a container and can stop any break within a box
  • Responsive typography: "em" is relative to the parent element's font-size whereas "rem" is relative to root font-size
  • Font options should consider legibility in relation to accessbility
  • Change cursor style
  • Margin vs padding: margin is the space outside of an element whereas padding is the space surrounding the content within an element
  • Uniform image sizing
  • Make a div invisible to clicks
  • Rotate an element
  • Use media queries and display:none or display:block to make divs appear/disappear at different screen widths
  • Increase scrolling smoothness
  • Box-sizing includes padding and border in the elements total height and width
  • To make an indent start from the second line, you can use a negative indent value to relatively shift the top line to the left
