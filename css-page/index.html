CSS

Cascading Style Sheets (CSS)

A language which dictates how HTML elements are to be displayed

Best practices

  • Comments: use comments to add a brief note or explanation to the code
  • Class-based styling
  • Accessibility:
    select fonts which are appropriate for low vision
    ensure sufficient contrast
    allow for magnification
    ensure sufficient spacing
  • Formatting: order the code in a logical order and with flagging comments for each section
  • Validate the code: use a CSS validator to check the code for errors

Hints

  • Check the spelling of each class
  • Check container levels
  • Check folder pathway
  • Check color and contrast
  • Check mobile site portrait and landscape
  • Google Fonts provides a range of free font options
